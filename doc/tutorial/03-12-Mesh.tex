\subsection{メッシュ}

メッシュはシーンの構成要素の１つでMeshクラスで扱います。
メッシュは頂点データを取り扱うVertexBufferクラスと
面を定義するインデックス配列のIndexBufferクラス、材質を定義するAppearanceクラスを持ちます。


\begin{verbatim}
Mesh* msh = new Mesh (vertices, submeshes, indices, apps);
wld->addChild (msh);
\end{verbatim}

\subsubsection{コンストラクタ}

頂点データと面を構成するインデックス配列およびアピアランスを指定してインスタンス化します。
１つのインデックス配列が１つのサブメッシュを定義し、１つまたは複数のサブメッシュを同時に指定できます。

\begin{verbatim}
 	Mesh (VertexBuffer *vertices, int num_submesh, IndexBuffer **submeshes, int num_appearance, Appearance **appearances)
 	Mesh (VertexBuffer *vertices, IndexBuffer *submesh, Appearance *appearance)
\end{verbatim}

現在のサブメッシュ数はgetSubmeshCount()メソッドで取得できます。

\begin{verbatim}
int 	getSubmeshCount () const
\end{verbatim}

現在のインデックス配列を取得するにはgetIndexBuffer()メソッドを使用します。

\begin{verbatim}
IndexBuffer * 	getIndexBuffer (int index) const
\end{verbatim}

引数のindexにはサブメッシュの番号を0から始まる整数値で指定します。

現在の頂点データを取得するにはgetIndexBuffer()メソッドを使います。

\begin{verbatim}
VertexBuffer * 	getVertexBuffer () const
\end{verbatim}

アピアランスは後からsetAppearance()メソッドを使って変更する事ができます。
NULLを指定するとデフォルトのアピアランスが使われます。

\begin{verbatim}
void 	setAppearance (int index, Appearance *appearance)
\end{verbatim}

第１引数のindexにはサブメッシュの番号を指定します。
第2引数のappearanceにはセットしたいAppearaceオブジェクトを指定します。
NULLを指定するとデフォルトのアピアランスが使われます。

サブメッシュに定義されているアピアランスを取得します。

\begin{verbatim}
Appearance * 	getAppearance (int index) const
\end{verbatim}

第１引数のindexにはサブメッシュの番号を指定します。
